# Sistema de Cadastro de Alunos

Este projeto consiste em um sistema de Cadastro de Alunos desenvolvido utilizando o framework Django. A aplicação permite realizar operações de CRUD (Create, Read, Update e Delete) para gerenciamento de alunos.

Além disso, o projeto possui uma API desenvolvida com Django REST Framework, que permite acessar os dados cadastrados no sistema e retorná-los no formato JSON.

## Funcionalidades

- Cadastro de alunos
- Listagem de alunos
- Atualização de dados dos alunos
- Exclusão de alunos
- API para consulta de dados em JSON

## Tecnologias utilizadas

- Python
- Django
- Django REST Framework
- HTML (Templates Django)
- SQLite

## Estrutura do projeto

O projeto está organizado em:

- Models: responsáveis pela estrutura dos dados
- Views: responsáveis pela lógica da aplicação
- Templates: responsáveis pela interface do sistema

## API

A API permite consultar os dados cadastrados no sistema através de endpoints que retornam respostas em formato JSON.
